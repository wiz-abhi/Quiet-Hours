## 📖 The Ideas In Detail

### 1. Quiet Hours ⭐
Nonprofit/civic on-call agent. Watches for "one person, alone, past midnight, still typing" → silently rate-limits non-critical pings, drafts handoff, wakes backup via MCP, posts morning "thank you" Canvas.
- **Killer demo:** 01:47am. Private DM: *"You've been on this alone for 3h12m. I've drafted a handoff, silenced 4 non-P0 pings, and paged @marcus. Sleep. I'll wake you if it escalates."* Cut to 08:04am Canvas titled *"What Priya saved last night."*
- **Tech:** RTS (solo-detection) + MCP (PagerDuty/Opsgenie handoff) + Canvas
- **Track:** Agent for Good

### 2. Fieldline
Trauma-aware coordination for crisis-response Slack workspaces. Detects "linguistic compression" (message length dropping = pre-collapse signal) → coordinates warm handoff via MCP-connected on-call tools.
- **Killer demo:** #wildfire-intake channel; responder "Marco"'s messages shorten to "copy", "k", "ok". Private DM: *"Marco — message length dropped 60%. Sam just came online. Want me to hand off?"* Text: *"Marco went home. He didn't have to ask."*
- **Adversarial pivot:** Reframe as **"fatigue dashboard with one-tap handoff"** — never "trauma detection." Kills paternalism attack.
- **Tech:** RTS + Slack AI + MCP · **Track:** Agent for Good

### 3. Marginalia
Book club companion syncs to members' Kindle/Kobo highlights. During Slack discussion, drops the exact passages other members underlined — anonymously.
- **Killer demo:** Book club discussing *Bluets*. Someone types *"I don't get why she keeps returning to that blue jar."* Marginalia posts a Canvas: *"Four of you underlined this passage. Two of you underlined it twice."* Anonymous reactions: *"—reminded me of my mother." "—I read this on the train home from the hospital."* Someone types: *"oh."*
- **Adversarial pivot:** Readwise-only, moderator-triggered reveal, target grief/hospice groups.
- **Tech:** MCP (Readwise) + RTS + Canvas · **Track:** New Slack Agent (pivot: Agent for Good)

### 4. Socratic Shadow
Study-group agent that **refuses to give answers** — instead replies with a Socratic counter-question that quotes the student's own past reasoning.
- **Killer demo:** Student asks *"why does my binary search return -1??"* Agent: *"Three days ago in #ds-help you told Priya that mid = (low+high)/2 overflows. What's low when high becomes 1?"* Student fixes it themselves in 40 seconds.
- **Adversarial pivot:** Pre-bake as scripted golden path (no live LLM roulette in video). Add visible "receipts" UI showing the retrieved quote.
- **Tech:** RTS + Slack AI · **Track:** Agent for Good

### 5. Blastradius
Paste a PR link → agent renders which teammates own the touched code, which recent Slack threads discussed those files, which on-call channels will page if it breaks.
- **Killer demo:** Dev drops PR into #frontend. Interactive dependency graph Canvas appears, red nodes pulse where diff touches code discussed in a #incidents thread 3 days ago — with Sarah's quote *"do NOT refactor this without asking me"* surfaced next to her avatar.
- **Tech:** RTS + MCP (GitHub) + Canvas · **Track:** New Slack Agent · **Best Tech Impl shot: 9.0** (highest)

### 6. Dungeon Master's Familiar
Tabletop RPG companion. Tracks every NPC across 200+ sessions. Generates in-character "letters from NPCs" between sessions.
- **Killer demo:** Monday morning Canvas titled *"A letter from Bram the Blacksmith"*: *"Aria — you left three months ago promising to return with the silver. The forge is cold. Elenwen asks about you every Tuesday."* Player had forgotten Bram. DM had too. Familiar didn't.
- **Adversarial pivot:** Reframe as "Session Zero Companion" — ingests last 4 sessions, tracks forward. Pre-render letters as PNG images (not Canvas markdown).
- **Tech:** MCP + RTS + Canvas · **Track:** New Slack Agent

### 7. Kintsugi
Private bereavement channel companion. Paces condolence messages, remembers the deceased, helps grieving family respond without drowning.
- **Killer demo:** 47 unread condolences. Kintsugi gathers them into a soft-gold Canvas, groups by relationship, drafts three reply options in the widow's own voice.
- **Track:** Agent for Good · **Risk:** Ethics minefield.

### 8. Nana
Slack for elderly parents. AI translates family group-chat noise into one calm daily "letter from the kids"; translates their voice memos back.
- **Killer demo:** 340 chaotic family messages → Grandma's iPad shows one card, one photo, three sentences, a big mic button. She replies by voice. Family sees a warm text reply in Grandma's actual voice pattern.
- **Track:** Agent for Good · **Risk:** Voice-clone consent surface + 7-day feasibility.

### 9. Ghostwriter's Trace
Students USE AI openly on essays; agent auto-generates a transparent timeline Canvas showing which ideas were theirs, which came from AI.
- **Killer demo:** `/trace-submit` → Canvas shows vertical timeline. Green dots = original ideas. Blue = AI suggestions. Red = one flagged verbatim paste. *"Originality contribution: 71%. Professor Chen has been sent this trace."*
- **Track:** Agent for Good · **Risk:** AI-integrity culture war divides judges.

### 10. Handoff
When you sign off on-call, agent writes a warm handoff letter to the next person in your voice.
- **Killer demo:** Canvas titled *"For Aiko, from Marcus — Tuesday night"*: *"Hey Aiko, the flaky test Devon mentioned at 2pm? Same Postgres race we saw in April. Also, please be gentle with Priya, she's had a rough day."*
- **Track:** Agent for Good · **Risk:** Essentially a template generator — Tech Impl cratered.

## 🎓 What The Analysis Taught Us

Four lessons that apply regardless of which idea you pick:

1. **Demo-fragility is the silent killer.** The three "beautiful moment" ideas (Marginalia, DMF, Kintsugi) all scored high on paper but the adversarial reviewers converged on the same attack: one bad take on video day = no fallback. Quiet Hours wins because it does something legible and useful even if the emotional beat doesn't land.

2. **Fake numbers are the loudest tell.** Judges from Salesforce/Slack watch hundreds of videos; they instantly clock fabricated impact metrics ("2,300 meal deliveries saved"). **Only cite what the agent legitimately observed.**

3. **RTS must be VISIBLE, not just load-bearing.** An RTS integration that judges can't see in the demo scores the same as one you didn't build. Show the retrieved evidence in a receipts card UI whenever possible.

4. **Track selection is a 30% edge before you write a line of code.** Agent for Good has meaningfully thinner competition and pre-loads your Impact score (25% of judging).
